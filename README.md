## 操作说明

- 数据库自动生成，按照提示下一步即可
- 默认用户：admin
- 默认密码：admin@123
- 不喜欢默认用户请自行注册用户
- 搜索功能均可用，直达百度
- 可以自己给自己发邮件

### 使用教程

- 安装小皮面板并进入小皮面板的WWW目录下

```bash
git clone git@github.com:xiaokai54/simple-resume.git
```

- 直接使用`localhost/simple-resume`即可
